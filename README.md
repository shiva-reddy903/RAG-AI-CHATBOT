# 🤖 Smart Doc Chatbot

A Retrieval-Augmented Generation (RAG) chatbot built using Python, Streamlit, LangChain, Groq, ChromaDB, and HuggingFace Embeddings.

## Features

- Upload PDF documents
- Chat with uploaded PDFs
- Semantic Search
- Chroma Vector Database
- Groq LLM
- Source Citation
- Chat History
- Streamlit UI

## Tech Stack

- Python
- Streamlit
- LangChain
- Groq
- ChromaDB
- HuggingFace Embeddings
- PyPDF

## Installation

```bash
git clone https://github.com/yourusername/SMART-DOC-CHATBOT.git

cd SMART-DOC-CHATBOT

pip install -r requirements.txt
```

Create a `.env` file:

```text
GROQ_API_KEY=your_api_key
```

Run the application:

```bash
streamlit run main.py

```
